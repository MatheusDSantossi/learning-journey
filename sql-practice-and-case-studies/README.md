# SQL Analysis

## 📌 Overview
This folder contains some SQL practice project based on a several dataset and use cases.  
The goal of this folder is to improve SQL skills through hands-on exercises and analytical queries.

The initial structure and questions are inspired by an existing open-source project, and this repository will be expanded over time with additional exercises, custom queries, and deeper analysis.

---

## 📊 Dataset
The dataset used in one this project contains information about Netflix titles, including:
- Type (Movie / TV Show)
- Title
- Director and cast
- Country
- Release year
- Date added to Netflix
- Rating
- Duration
- Genres
