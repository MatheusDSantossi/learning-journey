## Parte 3: VALUATION - MÉTODOS DE PRECIFICAÇÃO

### 3.1 Método de Fluxo de Caixa Descontado (DCF)

#### Premissas Base

```markdown
TAXA DE DESCONTO (WACC)
=======================
Custo de Capital Próprio (Ke): 8,5-9,0% (ao ano real)
    - Taxa Livre de Risco: ~3,0%
    - Beta: ~0,75
    - Prêmio de Risco: ~6,0%
    - Spread país (Brasil): ~1,0%

Custo de Dívida (Kd): 6,0% (nominal)
    - Ajustado por imposto: 4,5%

Estrutura de Capital
    - Valor de Mercado Equity: ~17,95B
    - Dívida Líquida: ~13,9B
    - Total: ~31,85B
    - % Equity: 56%
    - % Dívida: 44%

WACC = (56% x 9,0%) + (44% x 4,5%) = 6,96% (real)
WACC Nominal = 10,5% (incluindo infalção de 3,5%)
```

#### Preojção de Fluxos de Caixa Livres (FCL)

**Período 1: 2026-2028 (Ciclo de Expansão)**

| Ano | 2026e | 2027e | 2028e |
|-----|-------|-------|-------|

| **Crescimento** | 1,5% | 2,0% | 3,0% |

| **Margem EBITDA** | 36,6% | 37,1% | 37,8% |
| **Depreciação** | (1,85) | (1,90) | (1,95) |
| **EBIT** | 1,65 | 1,72 | 1,85 |
| **Imposto (34%)** | (0,56) | (0,58) | (0,63) |
| **NOPAT** | 1,09 | 1,14 | 1,22 |
| **+ Depreciação** | 1,85 | 1,90 | 1,95 |
| **- CapEx** | (4,20) | (3,80) | (3,20) |
| **- Var. Cap. Giro** | (0,20) | (0,15) | (0,10) |
| **= FCL** | (1,66) | (0,81) | (0,13) |

*Nota FCL negativo reflete pico de investimentos em 2026-2027*

**Período 2: 2029-2035 (Maturação)**

| Ano | 2029e | 2030-2035 | 2035e |
|-----|-------|----------|-------|
| **Receita Líquida** | 10,55 | IPCA + 2% | 12,40 |

| **Crescimento FCL** | | 4,0% a.a. | |
| **FCL Normalizado** | 0,80 | | 1,50 |

#### Cálculo do Valor Presente

```markdown
DCFF = Σ FCL / (1 + WACC)^t + VT / (1 + WACC)^n

Onde:
VT (Valor Terminal - Perpetuidade) = FCL (2035) x (1 + g) / (WACC - g) = 1,50 x 1,025 / (0,10 - 0,025) = 20,5B

Soma PV (2026-2028): -2,1B
PV Valor terminal: 14,5B
Enterprise Value: 12,4B

Menos: Dívida Líquida: (13,9B)
Equity Value: (1,5B)

Valor por Ação: Indefinido/negativo
```

⚠️ **CONCLUSÃO DCF:** O modelo puro de DCF apresenta desafios devido ao FCL negativo no período de investimentos. O valor é dependente de:

1. **Execução perfeita do plano de capex**
2. **Rentabilidade dos novos projetos (TIR > 9%)**
3. **Normalização da alavancagem pós-2027**

### 3.3 Método de Desconto de Dividendos (Dividend Discount Model - DDM)

```markdown
Para empresas pagadoras de dividendos estáveis:

P = D x (1 + g) / (Ke - g)

Onde:
D = Dividendo anual: R$ 1,22
g = Taxa de crescimento: 3,5% (IPCA + 0,5%)
Ke = Custo de capital próprio: 9,0%

P = 1,22 × 1,035 / (0,09 - 0,035)
P = 1,26 / 0,055 = R$ 22,91

INTERVALO DE SENSIBILIDADE:
- Ke 8,5%, g 3,0%: P = R$ 25,40
- Ke 9,0%, g 3,5%: P = R$ 22,91
- Ke 9,5%, g 4,0%: P = R$ 21,20
```

---

### 3.4 Método de Bazin (Fórmula do Décio Bazin)

Para empresas pagadoras de dividendos:

```markdown
P = D / TMA

Onde:
D = Dividendo médio histórico (últimos 5 anos)
TMA = Taxa Mínima Aceitável (6% para qualidade alta)

Estimativa D (últimos 5 anos): ~R$ 2,98
P = 2,90 / 0,06 = R$ 48,33
 
CONSIDERAÇÕES:
- Este método superestima porque inclui períodos com dividendos extraordinários (RBSE)
- Ajustado para dividendo normalizado: R$ 1,50
- P (ajustado) = 1,50 / 0,06 = R$ 25,00
```

---

### 3.5 Método de Graham (Fórmula de Segurança)

```markdown
P = (LPA × (8.5 + 2g) × 4.4) / Y
 
Onde:
LPA = R$ 2,46
g = Taxa de crescimento (3,5%)
Y = Yield de títulos (5,5%)

P = (2,46 × (8,5 + 7) × 4,4) / 5,5
P = (2,46 × 15,5 × 4,4) / 5,5
P = 167,87 / 5,5 = R$ 30,52
 
INTERVALO DE SEGURANÇA: R$ 30,52 × 0,75 = R$ 22,89
```
